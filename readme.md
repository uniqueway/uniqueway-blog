> 无二之旅技术博客

### 安装

``` bash
git clone https://github.com/uniqueway/uniqueway-blog
cd uniqueway-blog 
npm install hexo-cli -g
hexo server
```

### 撰写

``` bash
cd uniqueway-blog
hexo new [layout] <title>
hexo generate
git add .
git push origin master
```

[More Documentation](https://hexo.io/docs/commands.html)

### 部署

`hexo deploy`
